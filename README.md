# System Audio Recorder

`System Audio Recorder` **rec.exe** - небольшая утилита для Windows, которая записывает системный звук через loopback-захват.

Программа работает в системном трее, поддерживает горячие клавиши, сохраняет запись в `WAV` или `MP3` и показывает компактный экранный индикатор состояния с таймером.

## Возможности

- запись системного звука с устройства воспроизведения Windows по умолчанию
- запуск и остановка записи через горячие клавиши или меню в трее
- сохранение результата в `WAV` или `MP3`
- выбор частоты дискретизации из меню
- отображение компактного плавающего индикатора состояния
- встроенное окно справки

## Управление

Горячие клавиши:

- `Ctrl+Alt+R` - старт или остановка записи
- `Ctrl+Alt+O` - показать или скрыть индикатор состояния
- `Ctrl+Alt+Q` - полностью закрыть программу

Меню в трее:

- `Start` - начать запись
- `Stop` - остановить запись и сохранить файл
- `MP3 / WAV` - выбрать формат сохранения
- `9600 Hz ... 96000 Hz` - выбрать частоту дискретизации
- `Help` - открыть окно справки
- `Exit` - закрыть программу

## Ожидаемый результат работы

После запуска программа помещает иконку в системный трей Windows и показывает компактный плавающий индикатор на экране.

Во время записи:

- иконка в трее переключается в режим записи
- на индикаторе мигает красный круг
- таймер показывает длительность текущей записи

После остановки записи:

- аудио сохраняется в папку проекта как `system_YYYY-MM-DD_HH-MM-SS.wav` или `system_YYYY-MM-DD_HH-MM-SS.mp3`
- индикатор возвращается в состояние `STOP`

## Требования

- Windows
- Python 3.13 и зависимости из [`requirements.txt`]
- опционально: `ffmpeg` в `PATH`, если нужен экспорт в `MP3`

## Запуск

```powershell
venv\Scripts\python.exe rec.py
```

## Сборка

Если установлен `PyInstaller`, исполняемый файл можно собрать командой:
